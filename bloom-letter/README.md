## QRコード用URL

GitHub Pagesで公開後、以下のようにURL末尾を変えます。

- Aテーブル: `?flower=A`
- Bテーブル: `?flower=B`
- Cテーブル: `?flower=C`
- ...
- Nテーブル: `?flower=N`

例：
`https://あなたのユーザー名.github.io/bloom-letter/?flower=A`
